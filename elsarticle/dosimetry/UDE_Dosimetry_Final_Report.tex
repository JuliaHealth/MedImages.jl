\begin{abstract}
This report details the implementation and benchmarking of Universal Differential Equation (UDE) models for voxel-level dosimetry. We demonstrate that UDEs achieve high-fidelity radiation transport physics ($r = 0.957$) using $64^3$ voxel patches, significantly outperforming analytical baselines while maintaining clinical real-time performance.
\end{abstract}

\section{Model Architectures and Data Flow}

We implemented three primary architectures to evaluate the tradeoff between mechanistic constraints and black-box capacity. All models operate on 3D volumetric patches.

\subsection{Inputs, Outputs, and General Data Flow}
All models process three-dimensional anatomical and functional data to predict the final cumulative dose.
\begin{enumerate}
    \item \textbf{Anatomical Prior}: CT Hounsfield Units (HU) converted to physical mass density $\rho$ (g/cm$^3$).
    \item \textbf{Functional Prior}: [$^{177}$Lu]Lu-PSMA SPECT activity (Bq) providing initial radiopharmaceutical distribution.
    \item \textbf{Analytical Prior}: (Optional) Fast water-equivalent dosemap approximation $D_{approx}$.
\end{enumerate}

\paragraph{UDE Data Flow:} The system initializes a mechanistic pharmacokinetic (PK) model. At each time-step of the numerical integration, the current activity state and CT density are processed by a neural network to calculate the \textit{residual dose rate}. This rate is added to the physical local deposition rate and integrated over a 300-hour therapeutic window to produce the final 3D dosemap.

\paragraph{CNN Data Flow:} The SPECT, CT, and $D_{approx}$ maps are concatenated into a multi-channel 3D volume. The network performs a direct end-to-end mapping using a deep residual stack to predict the voxel-wise cumulative dose in a single forward pass.

\section{Performance Benchmarks}

The results presented below were obtained using **$64 \times 64 \times 64$ voxel patches**. It is important to note that performance trends were highly **consistent with $32 \times 32 \times 32$ patches**, although the larger $64^3$ spatial context was required to achieve peak correlations by capturing long-range scattering.

\section{Conclusion}
The **No-Approx UDE** architecture is the definitive production model for precision dosimetry. By matching the parameters of a deep CNN while maintaining physical consistency, it offers the most robust path for capturing heterogeneous radiation transport.
