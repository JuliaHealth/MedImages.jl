\begin{abstract}
This report details the final implementation and comprehensive benchmarking of Universal Differential Equation (UDE) models for voxel-level dosimetry. We demonstrate that UDEs achieve high-fidelity radiation transport physics ($r = 0.957$) using $64^3$ voxel patches, significantly outperforming analytical baselines, standard 3D CNNs, and established architectures including PBPK-ML, Spect0, DblurDoseNet, and SemiDose.
\end{abstract}

\section{Model Architectures and Data Flow}

We evaluated a comprehensive suite of architectures to benchmark our UDE framework against the current state-of-the-art in deep-learning and statistical dosimetry.

\subsection{Data Flow and Inputs}
All models process 3D volumetric patches ($64 \times 64 \times 64$) consisting of:
\begin{enumerate}
    \item \textbf{CT Density} ($\rho$): Standardized Hounsfield Units.
    \item \textbf{SPECT Activity} ($A_0$): Standardized radiopharmaceutical distribution.
    \item \textbf{Analytical Prior} ($D_{approx}$): (Used by UDE and Spect0) Fast water-equivalent approximation.
\end{enumerate}

\subsection{Baseline Architectures}
\begin{itemize}
    \item \textbf{PBPK-ML (Extra Trees)}: Adapted from the "Predictive Dosimetry via Machine Learning" (PBPK-TDT) framework. We implemented a voxel-wise Extra Trees regressor using five local features: intensity, density, local means (3x3x3), and Euclidean distance to high-uptake peaks (cross-fire proxy).
    \item \textbf{Spect0 (3D Residual)}: A 3D U-Net that learns a spatial correction to the analytical baseline.
    \item \textbf{SemiDose (3D Mean Teacher)}: A semi-supervised framework using consistency regularization.
    \item \textbf{DblurDoseNet (Plain 3D U-Net)}: A deep encoder-decoder model optimized for mapping blurred SPECT to sharp dose maps.
\end{itemize}
